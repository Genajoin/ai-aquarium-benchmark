# AI Aquarium Benchmark 🐠

A collection of terminal-based aquarium screensavers generated by different AI models. Each implementation create an animated underwater scene with fish, bubbles, seaweed, and corals.

## Overview

This project showcases how different AI models interpret and implement the same prompt:

> "Write a terminal screensaver in Python. Theme — aquarium. Bubbles, seaweed, fish, corals. ASCII art in motion."

Each `.py` file represents a unique implementation by a specific AI model, demonstrating variations in code style, animation techniques, and creative interpretation.

## Requirements

- Python 3.6+
- Unix-like terminal with curses support (Linux, macOS, WSL)

## Running

### Run all screensavers sequentially
```bash
./run_all.sh
```
Press **Enter** to run each file, or **ESC** to skip it.

### Run a single screensaver
```bash
python3 <filename>.py
```

### Exit any screensaver
Press any key or `Ctrl+C`.

## Features

Each implementation typically includes:

- **Fish** — Animated sprites swimming left/right, some with multi-line designs
- **Bubbles** — Rising from bottom to top with horizontal drift
- **Seaweed** — Swaying animation using sine waves
- **Corals** — Static or animated multi-line sprites at the bottom
- **Floor** — Decorative bottom row with animated characters
- **Water surface** — Animated top row
